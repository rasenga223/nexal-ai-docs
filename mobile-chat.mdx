## Starting Your First Conversation

Chatting with AI in Nexal is as simple as texting a friend, but much more powerful!

### How to Start a Chat

**On Mobile:**
1. Open the Chat tab
2. Tap the text input box at the bottom
3. Type your question or message
4. Tap the Send button
5. Watch the AI respond in real-time as it types

### Understanding AI Responses

- AI responses appear word-by-word (streaming) for a natural conversation feel
- Responses are formatted with proper paragraphs, code blocks, and structure
- You can copy, share, or regenerate any response
- Long-press on a message to see options

### Tips for Better Results

- **Be specific**: Instead of "tell me about dogs," try "what are the best dog breeds for families with children?"
- **Provide context**: "I'm planning a trip to Japan. What should I pack?"
- **Ask follow-up questions**: The AI remembers your conversation
- **Use examples**: "Write a professional email like this: [example]"

## Choosing the Right AI Model

### What Are AI Models?

Think of AI models as different experts with different specialties. Some are great at creative writing, others excel at coding, and some are perfect for analysis. Nexal AI gives you access to over 100 different AI models!

### How to Change Models

**On Mobile:**
1. In the Chat tab, tap the model name at the bottom of the page
2. Browse available models by category (Fast, Pro, Basic, Legacy)
3. Select the model you want to use
4. Your conversation continues with the new model

### Model Categories

- **Fast**: Quick responses, great for simple tasks
- **Pro**: Premium models with better quality and reasoning
- **Basic**: Standard models for general use
- **Legacy**: Older versions, still available if needed

## Using Personas

### What Are Personas?

Personas are pre-configured AI assistants designed for specific tasks. Instead of explaining what you want each time, personas give the AI a role and set of instructions automatically.

### How to Use a Persona

**On Mobile:**
1. In the Chat input, type `@` followed by the name of the persona you want to use
2. Browse available personas from the dropdown
3. Select a persona
4. Start chatting - the AI will automatically adopt that role!

### Creating Custom Personas

Want your own specialized assistant? Create a custom persona:

**On Mobile:**
1. Go to the Personas screen
2. Tap "Create New Persona"
3. Fill in name, description, and instructions
4. Upload an avatar (optional)
5. Save and use in any conversation

## Advanced Chat Features

### Web Search Integration

Need up-to-date information? Enable web search:

**On Mobile:**
1. The AI can automatically search the web when needed
2. Results are integrated into the response with citations
3. Great for current events, real-time data, and recent information

### Reasoning Controls

Control how much the AI thinks before answering. Low for quick facts, Medium for most things, High when you need deep analysis. It'll show you its reasoning process, which is great for understanding how it solved something or catching errors in logic.

### Rapid Mode

Type `!rapid` before your message to get faster responses. The AI skips the deep thinking and gives you the first good answer.

Good for simple questions. Not great for complex problems where you want a thorough answer.

### Document Uploads

Upload files directly in chat:

**On Mobile:**
1. Tap the attachment icon in chat
2. Choose your file type (image, document, PDF)
3. Select a file
4. The AI can read and analyze your document
5. Ask questions about the document content

**Supported File Types:**
- PDF documents
- Images (JPG, PNG, etc.)
- Text files

### Tools

Tools extend the AI's capabilities beyond text generation.

**On Mobile:**
- **Image Generation**: Create images directly from chat
- **Document**: Write and edit documents
- **Memory**: Save important information for future recall
- **Reminder**: Schedule reminders and notifications
- **Web Search**: Get real-time information from the internet
- **Folder Access**: Reference files from your project folders

To use tools, type `/` in the chat input to see available options, or just ask the AI naturally.

### Voice Conversations (Mobile)

Have hands-free conversations:

1. Tap the microphone icon in the chat
2. Speak your message naturally
3. The AI responds with both text and voice
4. Great for when you're on the go

Voice features include:
- Multiple voice options
- Real-time conversation flow
