Folders are a powerful way to organize your AI interactions and documents. In Nexal AI, folders aren't just for organization—they provide context that helps the AI understand your projects better.

## Understanding Folders

Think of a folder as a **dedicated project workspace**. When you create a folder, you can group related conversations and upload documentation that the AI can reference.

### Key Capabilities

- **Organize Chats**: Group related conversations together by project, topic, or client.
- **Project Files**: Upload files, images, and documents directly to a folder.
- **AI Context Awareness**: When chatting within a folder, the AI automatically understands the context and can search through all files in that folder.
- **Visual Previews**: Quick access to file thumbnails and document icons within the folder panel.

## Managing Your Folders

### Creating a Folder

1. Open the **Folders** panel from the sidebar or dashboard.
2. Click the **+ Create New Folder** button.
3. Enter a name and an optional description for your project.
4. Click **Create Folder** to save.

### Editing and Deleting

- **Edit**: Click the edit icon (pencil) next to a folder name to rename it or update the description.
- **Delete**: Click the trash icon to remove a folder. *Note: Deleting a folder will also remove the organization of conversations within it, but won't delete the conversations themselves.*

## Organizing Conversations

You can add any conversation to a folder to keep your sidebar clean and focused.

1. Hover over a conversation in your sidebar.
2. Click the **Move to Folder** option.
3. Select an existing folder or create a new one.
4. The conversation will now appear under that folder in the Folders panel.

## Managing Project Files

Folders allow you to store relevant documents right where you need them.

1. Open a folder in the **Folders** panel.
2. Click the **Upload file** button.
3. Drag and drop or select files from your computer.
4. Supported files include **PDFs, Images, Text documents, and more**.

### Interactive File Previews
Once uploaded, you can see thumbnails for images and distinct icons for different file types. Hovering over a file allows you to:
- **Delete**: Remove a file from the folder using the red trash icon.

## Folder-Aware AI

The real power of folders comes from **Context Awareness**. 

When you start a chat within a folder, the AI receives additional context about that folder. This enables the AI to:
- Use the **Knowledge Base Search** tool more effectively.
- Automatically reference files uploaded to that specific folder.
- Provide more relevant answers based on the project context.

<Info>
  **Pro Tip:** Create a folder for each of your major projects and upload all relevant background materials. When you chat in that folder, the AI will act like an expert on your specific project!
</Info>
