## Getting Better AI Responses

### Prompt Engineering Basics

The way you ask questions dramatically affects the quality of answers!

### Be Specific

- Bad: "Tell me about dogs"
- Good: "What are the best dog breeds for families with young children who live in apartments?"

### Provide Context

- Bad: "Write an email"
- Good: "Write a professional email to my manager requesting next Friday off for a family event"

### Use Examples

- "Write a professional email similar to this style: [paste example]"
- "Format this like a news article: [paste example]"

### Break Down Complex Requests

- Instead of one huge request, break it into steps
- Ask follow-up questions
- Build on previous responses

### Common Prompt Patterns

**For Writing:**
- "Write a [type] about [topic] in [style]"
- Example: "Write a blog post about healthy eating in a friendly, conversational style"

**For Analysis:**
- "Analyze [subject] and explain [aspect]"
- Example: "Analyze this data and explain the key trends"

**For Learning:**
- "Explain [concept] like I'm [age/level]"
- Example: "Explain quantum physics like I'm 10 years old"

**For Creative Projects:**
- "Create [type] with [elements] in [style]"
- Example: "Create a story with a robot and a cat in a sci-fi style"

## Productivity Tips

### Keyboard Shortcuts (Web)

- `Enter`: Send message
- `Shift + Enter`: New line
- `Cmd/Ctrl + K`: Search conversations
- `Cmd/Ctrl + /`: Show keyboard shortcuts

### Quick Actions

**Copy Response:**
- Long-press (mobile) or right-click (web) on any AI message
- Tap or click "Copy"
- Paste anywhere

**Regenerate Response:**
- Don't like an answer?
- Tap or click "Regenerate"
- Get a new version

**Edit and Resend:**
- Tap or click your message
- Edit it
- Resend for a new response

### Time-Saving Features

**Conversation Templates:**

**On Mobile:**
1. Go to Settings → Templates
2. Create templates for common questions
3. Quick access from chat input
4. Customize for different scenarios

**On Web:**
1. Go to Settings → Templates
2. Create templates for common questions
3. Quick access from chat input
4. Customize for different scenarios

**Voice Input:**
- Speak instead of type
- Faster for long messages
- Great for mobile use

**Quick Model Switching:**

**On Mobile:**
- Swipe on model name
- Quick switch between favorites
- No need to open menu

**On Web:**
- Click model selector
- Use arrow keys to navigate
- Press Enter to select

## Advanced Techniques

### Multi-Step Conversations

Break complex tasks into steps:

1. "First, help me understand [concept]"
2. "Now, apply that to [situation]"
3. "Finally, create a plan for [action]"

### Iterative Refinement

Start broad, then refine:

1. "Give me ideas for [topic]"
2. "Expand on idea #3"
3. "Make it more specific to [context]"
4. "Add [additional element]"

### Context Building

Provide context gradually:

1. "I'm working on [project]"
2. "The goal is [objective]"
3. "The constraints are [limitations]"
4. "Help me [specific task]"

### Combining Features

Use multiple features together:

- Upload a document + ask questions
- Use a persona + specific task
- Generate images + use in chat
- Create custom model + use persona

## Mobile-Specific Tips

### Voice Input Best Practices

- Speak clearly and at a moderate pace
- Use punctuation words: "period," "comma," "question mark"
- Review transcribed text before sending
- Works best in quiet environments

### Mobile Gestures

- Swipe to switch models
- Long-press for quick actions
- Pull down to refresh
- Swipe to delete conversations

### Offline Capabilities

- Some features work offline (viewing saved content)
- Chat requires internet connection
- Download important content for offline access

## Web-Specific Tips

### Browser Extensions

- Use browser extensions for quick access
- Pin the web app for easy access
- Use multiple tabs for different conversations

### Window Management

- Use split screen for multitasking
- Open multiple chat windows
- Organize tabs by project or topic

### Keyboard Navigation

- Use Tab to navigate
- Arrow keys for selections
- Escape to close modals
- Learn all shortcuts with `Cmd/Ctrl + /`

## Organization Tips

### Conversation Management

- Use folders to organize by project
- Archive old conversations
- Delete conversations you no longer need
- Search your history regularly

### Naming Conventions

- Use descriptive conversation names
- Include dates for time-sensitive topics
- Tag conversations with keywords
- Create consistent naming patterns

### Workflow Optimization

- Create templates for repetitive tasks
- Use personas for consistent results
- Save successful prompts
- Build a library of useful responses
