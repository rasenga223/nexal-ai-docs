## What Are Tools?

Tools extend the AI's capabilities beyond text generation, allowing it to perform specific actions and tasks. They enable the AI to interact with external systems and execute functions that go beyond conversation.

## Available Tools

### Web Tools

**Resume:**
- Generate professional resumes from your information
- Create formatted resume documents

**Image Generation:**
- Create images directly from chat using text descriptions
- Generate visuals without leaving the conversation

**Document:**
- Write and edit documents in DOCX format
- Use a built-in document editor
- Create and format documents within the chat interface

**Memory:**
- Save important information that the AI can recall in future conversations
- Build a persistent knowledge base for personalized interactions

### Mobile Tools

**Image Generation:**
- Create images directly from chat using text descriptions
- Generate visuals on the go

**Memory:**
- Save important information that the AI can recall in future conversations
- Access saved information across sessions

**Reminder:**
- Schedule reminders and notifications for important tasks
- Set up alerts and notifications

## Using Tools

To use tools, type `/` in the chat input to see available options, or mention the tool in your conversation. The AI will automatically use the appropriate tool when needed.

<Tip>
Tools are automatically selected by the AI based on your request, but you can also explicitly mention which tool you want to use.
</Tip>
