## What Are Tools?

Tools extend the AI's capabilities beyond text generation, allowing it to perform specific actions and tasks. They enable the AI to interact with external systems, generate content, and execute functions that go beyond conversation.

## Available Tools

### Web Tools

**Resume Generator**
- Generate professional resumes from your information
- Create formatted resume documents
- Export in various formats

**Image Generation**
- Create images directly from chat using text descriptions
- Generate visuals without leaving the conversation
- Multiple style options available

**Document Generator**
- Write and edit documents in DOCX format
- Use a built-in document editor
- Create reports, letters, proposals, and more
- Format and structure documents within the chat interface

**Memory**
- Save important information that the AI can recall in future conversations
- Build a persistent knowledge base for personalized interactions
- The AI automatically recalls relevant memories when needed

### Mobile Tools

**Image Generation**
- Create images directly from chat using text descriptions
- Generate visuals on the go
- Access to multiple AI image models

**Memory**
- Save important information that the AI can recall in future conversations
- Access saved information across sessions
- Persistent across all your chats

**Reminder**
- Schedule reminders and notifications for important tasks
- Set specific dates and times
- Choose repeat options (none, daily, weekly, monthly)
- Receive push notifications when reminders are due

**Web Search**
- Search the web for real-time information
- Get current news, events, and data
- Results are integrated directly into AI responses
- Citations included for source verification

**Folder Access**
- Reference files from your project folders in chat
- Upload and organize documents, images, and files
- The AI can read and analyze folder contents

## Using Tools

### Automatic Tool Use

The AI automatically selects the right tool based on your request. Just describe what you need:

- "Create an image of a sunset over mountains" → Uses image generation
- "Remember that my favorite color is blue" → Uses memory tool
- "Remind me to call mom tomorrow at 3pm" → Uses reminder tool (mobile)
- "What's the weather in Tokyo right now?" → Uses web search

### Manual Tool Selection

To explicitly choose a tool, type `/` in the chat input:

**On Web:**
1. Type `/` in the chat input
2. Browse available tools from the dropdown
3. Select the tool you want to use
4. Provide the required information

**On Mobile:**
1. Type `/` in the chat input
2. Browse available tools from the dropdown
3. Tap to select a tool
4. The AI will guide you through using it

### Tool Triggers

Quick triggers for common tools:

**On Web:**
- `!web` followed by space: Enable web search for your message
- `!rapid` followed by space: Use rapid mode for faster response

**On Mobile:**
- `@persona_name`: Use a specific persona
- `/tool_name`: Use a specific tool

## Tool Details

### Memory Tool

The memory tool helps the AI remember important information about you:

**What to Save:**
- Personal preferences
- Important dates
- Project details
- Recurring topics you discuss

**How It Works:**
1. Tell the AI to remember something: "Remember that I prefer Python over JavaScript"
2. The AI saves this to your memory
3. In future conversations, the AI recalls relevant memories automatically

**Managing Memories:**

**On Mobile:**
1. Go to the Memories screen
2. View all saved memories
3. Delete memories you no longer need

**On Web:**
1. Access memories from Settings
2. Review and manage saved information

### Reminder Tool (Mobile Only)

Schedule reminders with push notifications:

**Setting a Reminder:**
- "Remind me to submit the report on Friday at 2pm"
- "Set a reminder for my doctor's appointment next Tuesday"

**Reminder Options:**
- One-time reminders
- Repeating reminders (daily, weekly, monthly)
- Custom dates and times
- Timezone support

**Managing Reminders:**
1. View all reminders in the Reminders section
2. Mark reminders as complete
3. Edit or delete upcoming reminders

### Document Generator (Web Only)

Create professional documents directly in chat:

**Document Types:**
- Reports
- Letters
- Proposals
- Essays
- Meeting notes
- And more

**How to Use:**
1. Ask the AI to create a document: "Write a project proposal for..."
2. The AI generates the document
3. View and edit in the built-in editor
4. Export as DOCX

### Resume Generator (Web Only)

Create professional resumes:

1. Tell the AI about your experience and skills
2. The AI generates a formatted resume
3. Review and edit as needed
4. Export and download

## Tips for Tool Use

- **Be specific**: Clear instructions get better results
- **Provide context**: The more information you give, the better the output
- **Iterate**: Ask for revisions if the first result isn't perfect
- **Combine tools**: Use multiple tools in a single conversation for complex tasks

<Tip>
Tools are automatically selected by the AI based on your request, but you can also explicitly mention which tool you want to use for more control.
</Tip>
